# aopu

git批量发布工具
---

## 自动化发布
![publish.gif](https://github.com/mosikoo/blog/blob/master/assets/publish.gif?raw=true)

## Install

```bash
$ npm i aopu -g
```

## Usage

```bash
$ aopu --help
```

* 批量发布：在你所要发布项目的父目录执行`aopu`，然后根据提示进行操作
* 单独发布：在你项目目录中执行`aopu`, 然后根据提示进行操作
